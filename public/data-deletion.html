Instruções para Exclusão de Dados

Última atualização: 23/06/2026

O aplicativo InstaBot Ofertas pode armazenar dados operacionais recebidos por meio das APIs da Meta/Instagram, como nome de usuário, ID do comentário, ID da mídia, texto do comentário, data do evento e histórico de processamento da mensagem.

Como solicitar a exclusão

Para solicitar a exclusão dos seus dados relacionados ao aplicativo, envie um e-mail para:

skyvisionproducoes@gmail.com

Use o assunto:

Exclusão de dados - InstaBot Ofertas

No corpo do e-mail, informe o nome de usuário do Instagram relacionado à solicitação.

Processamento da solicitação

Após recebermos a solicitação, analisaremos os registros relacionados e removeremos os dados aplicáveis do banco de dados operacional dentro de prazo razoável, conforme requisitos legais, técnicos e de segurança.

Callback de exclusão de dados

Este aplicativo também possui endpoint técnico para solicitações de exclusão de dados:
