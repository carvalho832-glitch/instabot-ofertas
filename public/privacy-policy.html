Política de Privacidade

Última atualização: 23/06/2026

Esta Política de Privacidade descreve como o aplicativo InstaBot Ofertas, usado pela conta profissional do Instagram @minadeofertas_ofc, coleta, utiliza, armazena e protege dados recebidos por meio das APIs da Meta/Instagram.

1. Dados coletados

O aplicativo coleta apenas os dados necessários para operar a automação de comentários e mensagens do Instagram, incluindo:

  • nome de usuário do Instagram;
  • ID do usuário do Instagram, quando disponível;
  • ID do comentário;
  • ID da mídia, post ou Reel;
  • texto do comentário;
  • data e horário do evento;
  • status de processamento e envio de mensagem;
  • dados técnicos de webhook necessários para auditoria e depuração.

2. Como os dados são usados

Os dados são usados exclusivamente para identificar comentários com palavras-chave em publicações do Instagram, localizar o produto cadastrado no sistema e enviar uma resposta privada com informações da oferta, como nome do produto, preço, cupom e link de compra.

O aplicativo não vende dados pessoais, não compartilha dados com terceiros para publicidade e não utiliza os dados para finalidades não relacionadas ao atendimento solicitado pelo usuário.

3. Compartilhamento e operadores

Para funcionamento técnico do aplicativo, utilizamos provedores de infraestrutura e armazenamento em nuvem, incluindo Render para hospedagem do backend e Supabase para banco de dados operacional. Esses provedores processam dados somente para executar, armazenar e manter o serviço.

4. Retenção dos dados

Os dados são mantidos apenas pelo tempo necessário para histórico operacional, suporte, segurança, auditoria e depuração do sistema. Quando não forem mais necessários, poderão ser excluídos.

5. Exclusão de dados

Usuários podem solicitar a exclusão de seus dados relacionados ao app. Para isso, acesse a página de instruções de exclusão de dados:

https://instabot-ofertas.onrender.com/data-deletion.html

6. Segurança

Adotamos medidas técnicas e organizacionais razoáveis para proteger os dados processados pelo aplicativo, incluindo uso de variáveis de ambiente para tokens e armazenamento operacional em provedores de nuvem.

7. Contato
