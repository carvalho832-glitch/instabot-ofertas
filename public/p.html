Política de Privacidade

Última atualização: 20/06/2026

O aplicativo InstaBot Ofertas pode receber dados de interações públicas no Instagram, como nome de usuário, identificador do comentário, identificador da mídia, texto do comentário e horários de eventos enviados pela Meta.

Essas informações são usadas para identificar comentários de interesse, localizar produtos cadastrados e preparar atendimento automatizado sobre ofertas.

Não vendemos dados pessoais. Os dados são usados apenas para funcionamento do bot e podem ser armazenados em banco de dados privado configurado pelo responsável do aplicativo.

Para solicitar exclusão de dados, acesse Instruções para Exclusão de Dados.

Para dúvidas, use o e-mail de contato cadastrado no aplicativo Meta Developers.
