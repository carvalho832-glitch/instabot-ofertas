Instruções para Exclusão de Dados

Última atualização: 20/06/2026

O aplicativo InstaBot Ofertas pode armazenar dados de interação necessários para funcionamento do bot, como nome de usuário, identificador de comentário, identificador da mídia, texto do comentário e histórico de atendimento automatizado.

Como solicitar exclusão

Para solicitar a exclusão dos seus dados, envie uma solicitação para o e-mail de contato cadastrado no aplicativo Meta Developers com o assunto:

Exclusão de dados - InstaBot Ofertas

Inclua na mensagem o nome de usuário do Instagram relacionado à solicitação.

Prazo

Após recebermos a solicitação, faremos a análise e removeremos os dados aplicáveis em prazo razoável, conforme disponibilidade operacional.

Dados removidos

Podem ser removidos registros de comentários, interações simuladas ou reais, histórico de mensagens geradas e demais informações vinculadas ao usuário solicitante, quando tecnicamente possível.

Contato

Use o e-mail de contato configurado no aplicativo Meta Developers para enviar solicitações relacionadas a privacidade e exclusão de dados.
